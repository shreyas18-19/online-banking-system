# Online Banking System

A Full Stack Online Banking System developed using Java Servlets, JDBC, MySQL, and Apache Tomcat.

## Features

- User Registration & Login
- Secure Session Management
- Deposit Money
- Withdraw Money
- Transfer Money
- Transaction Receipt
- Transaction History
- Monthly/Yearly Bank Statement
- Card Services & Card Application
- Admin Approval Panel
- Forgot Password & Change Password

## Technologies Used

- Java
- Servlets
- JDBC
- MySQL
- HTML
- CSS
- Apache Tomcat

## Project Structure

online-banking-system
│
├── src
├── webapp
├── lib
└── db

## Database

Import the SQL file from:

db/banking.sql

## How to Run

1. Install MySQL and Apache Tomcat
2. Import database SQL file
3. Configure DBConnection.java
4. Compile Java files
5. Deploy project in Tomcat
6. Open browser:

http://localhost:8080/onlinebank/login.html
